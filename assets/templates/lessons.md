# 교훈 대장 — 반복되면 스킬로 승격한다

`/vault-session-end`가 매 세션의 교훈을 여기 축적하고, 같은 교훈이 **3회 반복되면 스킬 승격을 제안**한다 (hermes-agent 자기개선 루프의 파일 기반 구현).

## 규칙

- 새 교훈은 목록 **최상단**에 추가한다. 기존 교훈과 **같은 뜻**이면 새 줄을 만들지 말고 그 줄의 횟수를 올리고 근거를 덧붙인다 (의미 중복 = 이 대장의 적).
- 횟수 3 도달 시 세션 종료 보고에서 승격을 제안한다. **자동 승격 절대 금지 — 사용자 승인 후에만** 반영한다. 승격 대상 3종: ① CLAUDE.md 행동 계약 조항(규칙형) ② 새 슬래시 명령(절차형) ③ 기존 명령·스킬 문서 보강(수정형).
- 승격되면 상태를 `승격완료(→대상)`으로 바꾸되 줄은 남긴다(이력). 사용자가 기각하면 `기각`으로 바꾸고 다시 제안하지 않는다.
- 교훈의 자격: **세션을 횡단해 유효한 일반 원칙**만. 일회성 수정·당면 작업 상태는 교훈이 아니다.

형식: `- YYYY-MM-DD | 횟수 N | 교훈 한 줄 | 근거: 세션·커밋 | 상태: 관찰중|승격제안|승격완료(→대상)|기각`

## 대장

- (아직 교훈이 없다 — 첫 /vault-session-end가 채운다)
